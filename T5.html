📘 T1 - Introdução ao CSS

O que é CSS?

CSS significa Cascading Style Sheets. É a linguagem usada para estilizar páginas web.

Com CSS podemos mudar cores, fontes, tamanhos, espaçamentos, layouts e tornar o site moderno.

Sem CSS os sites ficariam apenas com texto simples e sem organização visual.

Importância do CSS

✔ Melhora o design do site

✔ Organiza os conteúdos

✔ Torna o site responsivo

✔ Separa conteúdo (HTML) da aparência (CSS)

✔ Permite criar animações e efeitos modernos

Tipos de CSS

1. Inline: Dentro da própria tag HTML.

2. Interno: Dentro da tag <style>.

3. Externo: Em ficheiro separado (.css).

O mais recomendado é o CSS Externo porque deixa o código mais organizado.

🎯 T2 - Seletores CSS

Seletores servem para escolher quais elementos HTML queremos estilizar.

Seletor de Elemento

Seleciona todas as tags iguais.

p {
  color: blue;
}

Seletor de Classe

Seleciona elementos com class específica.

.destaque {
  color: red;
}

Seletor de ID

Seleciona um único elemento.

#titulo {
  color: green;
}
