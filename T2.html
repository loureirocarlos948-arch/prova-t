Selectores em CSS

  • O quê são seletores em CSS?

    • Os seletores são usados para "selecionar" elementos HTML que você quer modificar com regras de estilo. Exemplo básico: p { color: blue;} Aqui, o seletor p seleciona todas as tags

    da página e define a cor do texto como azul.

    Tipos de Seletores CSS

  • 1. Seletor de Tipo (ou de Tag)

    • Seleciona todas as tags HTML de um tipo específico. h1 { color: red;} p { font-size: 18px;}

  • 2. Seletor de Classe (.classe)

    • Seleciona elementos com um atributo class.

    Texto em destaque

    .destaque { color: blue; font-weight: bold; } É usado com um ponto (.) antes do nome da classe. Vários elementos podem compartilhar a mesma classe.

  • 3. Seletor de ID (#id)

    • Seleciona um elemento com um identificador único.

    Título Principal

    #titulo { color: green; text-transform: uppercase; } É usado com uma hashtag (#). Cada página deve ter apenas um elemento com o mesmo ID.

  • 4. Seletor Universal (*)

    • Seleciona todos os elementos da página. * { margin: 0; padding: 0; box-sizing: border-box; } Muito usado para resetar o estilo padrão do navegador.

  • 5. Seletor de Agrupamento

    • Permite aplicar o mesmo estilo a vários seletores. h1, h2, h3 { color: darkblue; font-family: Arial, sans-serif; } Evita repetição de código.

  • 6. Seletor Descendente

    • Seleciona elementos dentro de outro elemento.

    Texto dentro da div

    div p { color: purple; } Aplica o estilo somente ao

    que está dentro da

  • 7. Seletor Filho Direto (>)

    • Seleciona apenas os filhos diretos de um elemento.

    Filho direto

    Filho indireto

    div > p { color: red; } Somente o primeiro

    será afetado.

  • 8. Seletor de Atributo

    • Seleciona elementos com base em atributos HTML. input[type="email"] { border: 2px solid green; } Também aceita variações: • [attr^=valor] → começa com valor • [attr$=valor] → termina com valor • [attr*=valor] → contém o valor Aplica estilos a estados específicos de um elemento. a:hover { color: red; } input:focus { border-color: blue; } Exemplos: • :hover → quando o mouse passa sobre • :focus → quando o campo está em foco • :first-child, :last-child → primeiro ou último filho

  • 9. Seletor Pseudo-classe (:)

    • Aplica estilos a estados específicos de um elemento. a:hover { color: red; } input:focus { border-color: blue; }
